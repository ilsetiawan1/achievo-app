# 🚀 Achievo — Daily Task Tracking with Gamification

## 📌 Overview

Achievo is a full-stack web application designed to help users manage daily tasks with a gamification approach. Users can track their tasks, set time durations, and earn experience points (EXP) based on their performance.

---

## ✍️ Description

Built a full-stack web application to manage daily tasks with a gamification system, featuring time-based task tracking and experience (EXP) rewards to enhance user productivity.

---

## 🛠️ Tech Stack

* Next.js (App Router)
* React
* MongoDB Atlas
* Mongoloquent (ORM)
* Tailwind CSS
* Heroicons

---

## 📁 Project Structure

```
src/
  app/
    api/
      auth/
        login/
          route.ts
        register/
          route.ts
      tasks/
        route.ts          # GET (all), POST (create)
        [id]/
          route.ts        # GET (detail), PUT (update), DELETE (delete)

  components/
    ui/                  # Button, Input, etc
    layout/              # Navbar
    task/                # TaskItem, TaskList, ExpBar

  hooks/
    useAuth.ts
    useTasks.ts

  lib/
    auth.ts              # JWT helper
    mongodb.ts           # DB connection

  models/
    User.ts
    Task.ts

  types/
    index.ts

.env                    # Environment variables
```

---

## 🔐 Authentication Flow

1. User registers or logs in
2. Server returns JWT token
3. Token is stored on client (cookie/localStorage)
4. Every API request includes token
5. Backend verifies token and extracts `userId`

---

## 🔁 Application Flow

```
Login / Register
      ↓
Get Token (JWT)
      ↓
Access Dashboard
      ↓
Create / Update / Delete Task
      ↓

      ↓

      ↓
Update Progress Bar
```

## 🧩 Features

### ✅ Authentication

* Register & Login
* JWT-based session
* User-specific data access

### ✅ Task Management

* Create task
* Update task
* Delete task
* Mark task as completed

### ⏱️ Time-Based Task

* Set start and end time
* Track completion time
* Detect late completion

### 🎮 Gamification (EXP System)

* On-time completion → full EXP
* Late completion → reduced EXP
* EXP calculated per task

### 📊 Progress Tracking

* Daily EXP accumulation
* Visual progress bar (0–100%)
* Real-time updates
